# Pharmacy Retail Analytics Dashboard

Interactive dashboard for hospital pharmacy sales analysis (2015 data). Built with Next.js 15 (App Router), TypeScript, Recharts, and shadcn/ui.

## Pages

| Page                | Route          | Purpose                                                   |
| ------------------- | -------------- | --------------------------------------------------------- |
| Executive Overview  | `/`            | Overall pharmacy performance, monthly trends, revenue mix |
| Product Performance | `/products`    | Generic vs branded analysis, SKU scatter, top 20 table    |
| Margin Risk         | `/margin-risk` | SKU margin analysis with interactive threshold slider     |

## Development

```bash
npm install
npm run dev        # localhost:3000
npm run build      # Static export to out/
npm run lint       # ESLint check
```

## Data

Static JSON files in `public/data/` are generated by the ETL pipeline (`etl/export_json.py`):

- `overview.json` — Monthly revenue, transactions, margin by channel
- `products.json` — Product type revenue, SKU scatter, top 20 SKUs
- `margin_risk.json` — All SKU margins with histogram distribution

## Deployment

Built for static export (`output: "export"` in `next.config.mjs`). Deploy to Cloudflare Pages, Vercel, or any static host.

```bash
npm run build
# Upload dashboard/out/ to your host
```
